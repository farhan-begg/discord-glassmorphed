## Getting Started
Follow these instructions to run the web application locally

### Prerequisites
* npm 

`npm install npm@latest -g`

or

* yarn

`brew install yarn`

### Installation
1. Clone the repo
https://github.com/farhan-begg/discord-glassmorphed.git

2. Install NPM packages

`npm install`

`npm start` or  `yarn start`
